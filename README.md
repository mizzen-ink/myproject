# 📂 myproject
> 基于 C++ 实现的客户端-服务器文件传输系统，支持用户身份验证与可靠文件传输

---

## ✨ 项目介绍
本项目采用经典 **C/S 架构（客户端-服务器）** 开发，实现了一个简易但完整的文件传输系统。
用户需通过账号密码登录验证后，才能与服务器进行文件交互，适合作为**网络编程、数据库应用**的课程设计或实战练习。

---

## 🚀 核心功能
- 🔐 **用户登录验证**：基于 MySQL 数据库存储账号信息，确保访问安全
- 📤 **文件上传**：客户端向服务器上传文件
- 📥 **文件下载**：客户端从服务器下载文件
- 🌐 **可靠传输**：基于 TCP Socket 实现稳定数据传输
- 🧩 **模块化设计**：代码结构清晰，易于维护与扩展

---

## 🛠️ 技术栈
| 技术 | 说明 |
|------|------|
| 编程语言 | C++ |
| 网络通信 | Linux Socket TCP |
| 数据库 | MySQL（C API 连接） |
| 运行环境 | Linux / WSL |
| 构建工具 | Makefile |
